# Release notes v2.17.0
**Datum:** 2026-06-27
**Type:** Minor

Er is nu bezoekersstatistiek ingericht: de website houdt bij welke mixen bezoekers bekijken.

## Features

### Bezoekersstatistieken voor mix-pagina's

Er is een Google Analytics-koppeling ingericht die automatisch registreert wanneer iemand een mix-pagina bezoekt, inclusief informatie zoals titel, kleur, genre en volume-nummer. Dit geeft inzicht in welke mixen het meest bekeken worden.

## Docs aanpassingen

### Werkafspraken vastgelegd

De projectrichtlijnen zijn bijgewerkt: een release wordt voortaan altijd op initiatief van de siteverantwoordelijke gestart, en afgeronde werkbranches worden direct na samenvoeging opgeruimd.
