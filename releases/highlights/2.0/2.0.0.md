# Release notes v2.0.0
**Datum:** 2026-03-07
**Type:** Major

De website van DJ Cylow is voor het eerst live gegaan.

## Features

### De website is live

Bezoekers kunnen de website nu bezoeken: een homepage, een overzicht van alle mixes gefilterd op sfeer, een detailpagina per mix met tracklist, en een luisterpagina met ingebouwde audiospeler.

---

### Contactformulier

Bezoekers kunnen via een contactformulier met ingebouwde spambescherming rechtstreeks een bericht sturen, dat automatisch per e-mail binnenkomt.

---

### Snelheid en vindbaarheid

De website scoort het maximale op snelheid, toegankelijkheid en vindbaarheid in Google.
