# Release notes v2.16.0
**Datum:** 2026-06-25
**Type:** Minor

Alle mixen hebben nu zoektags gekregen, wat mixen makkelijker vindbaar maakt.

## Features

### Zoektags toegevoegd aan alle mixen

Elke gepubliceerde mix (77 in totaal) heeft nu een set zoektags gekregen, vergelijkbaar met tags op YouTube. De tags zijn gebaseerd op subgenre, sfeer/kleur, jaar en de artiesten in de tracklist, en maken mixen beter vindbaar.
