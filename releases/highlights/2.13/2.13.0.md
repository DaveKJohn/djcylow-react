# Release notes v2.13.0
**Datum:** 2026-06-18
**Type:** Minor

Achter de schermen is de opslag van tracklijsten opgeschoond, wat de mix-pagina's stabieler en betrouwbaarder maakt. Voor bezoekers verandert er niets zichtbaars.

## Features

### Tracklijsten beter gestructureerd

De tracklijsten van mixen zijn omgezet van losse tekst naar een nette, gestructureerde vorm met aparte velden per track (artiest, titel, tempo, enzovoort). Dit maakt de mix-pagina's minder foutgevoelig, bijvoorbeeld wanneer een tracklijst-item ontbreekt.

---

### Algemene opschoning

De onderliggende code van de website is opgeschoond voor betere leesbaarheid en onderhoudbaarheid op de lange termijn.
