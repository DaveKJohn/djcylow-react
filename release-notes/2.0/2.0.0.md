**Datum:** 2026-03-07

## Wat is er live gegaan?

Dit is de eerste publieke versie van de DJ Cylow website, gebouwd met **Next.js 16** en
gedeployed via **Netlify**. Alles wat hier staat was aanwezig bij de livegang.

### Functionaliteit

- **Homepagina** — introductie met DJ Cylow branding en navigatie
- **Mixes-overzicht** — filterpagina met alle beschikbare mixes per genre/sfeer
- **Mix-detailpagina** — info, tracklist en luisterfunctie per mix
- **Luisterpagina** — geïntegreerde audiospeler voor mixes
- **Contactformulier** — met Google reCAPTCHA v2 validatie en e-mailverzending via Nodemailer
- **Lighthouse-score 100%** — geoptimaliseerd voor performance, accessibility, best practices en SEO

### Technische basis

- Framework: Next.js 16 (React 19)
- Styling: SCSS + Tailwind CSS 4
- Deployment: Netlify (`netlify.toml` geconfigureerd)
- Beveiliging: security headers (X-Frame-Options, X-Content-Type-Options, Referrer-Policy)
