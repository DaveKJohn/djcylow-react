# v1.14.2 — Script: nieuwe mix toevoegen

**Datum:** 2026-06-25
**Type:** Patch
**Branch:** `feature/add-mix-script`

## Wijzigingen

### Tooling

- `scripts/add-mix.js` toegevoegd: interactief script voor het toevoegen van een nieuwe mix
- `npm run mix:add` — start het script

### Wat het script doet

1. Vraagt interactief: kleur, power, frequentie, genre, datum, subgenre, BPM, volume, beschrijving en tracklist
2. Genereert automatisch: `id`, `title`, `jaar/maand/dag`, `permalink`, `audioSrc`, image paths
3. Stelt het volgende volumenummer voor op basis van bestaande mixes
4. Toont een preview van het volledige JSON object ter bevestiging
5. Voegt de mix bovenaan het juiste JSON bestand in (nieuwste eerst)

### Gebruik

```bash
npm run mix:add
```
