# v1.14.0 — Mix beschrijvingen alle kleuren + Red image update

**Datum:** 2026-06-25
**Type:** Content
**Branch:** `data/mix-descriptions-all-colors`

## Wijzigingen

### Mix beschrijvingen

- Unieke Nederlandse beschrijvingen toegevoegd aan alle eerder lege `description` velden in 14 JSON bestanden (alle kleuren, zowel Full als Light)
- Elke beschrijving bevat subgenre, sfeer en 2–4 artiesten (120–160 tekens, conform schema)
- `light-red.json`: generieke placeholder-teksten vervangen door unieke beschrijvingen per mix
- `light-red.json`: genre/subgenre gecorrigeerd bij Vol. 1 (`EDM / Drum & Bass` → `Drum & Bass / Neurofunk`)

### Afbeeldingen — Red (grote variant)

- Grote `.webp` afbeeldingen vervangen door `.jpg` voor alle Red mixes (`full/red/wide` en `light/red/wide`)
- Betere compatibiliteit met image-verwerkingstools; `.jpg` vervangt de eerder ontbrekende `.webp` bestanden

### Documentatie

- `CLAUDE.md` toegevoegd: projectinstructies voor Claude Code (git-workflow, veiligheidsregels, JSON-schema)
- `src/data/mixes/README.md` toegevoegd: volledige schema-specificatie voor mix JSON bestanden
